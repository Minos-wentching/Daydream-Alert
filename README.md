# Daydream-Alert
When classmates are in class or doing homework, they often do other things (such as scratching their heads,sleep, reading novels, etc.). In order to help students focus more on doing one thing, I plan to develop a daydream alert project to implement detection and issue alerts in a timely manner when students are distracted.
